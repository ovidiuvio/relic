API Documentation

Command Line Usage

# Basic usage
echo "Hello World" | curl -X POST https://api.Relic.dev/v1/pastes \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: text/plain"

# Advanced usage with options
cat logfile.txt | curl -X POST https://api.Relic.dev/v1/pastes \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Log File",
    "syntax": "text",
    "expiry": "24h",
    "visibility": "public"
  }'

REST API Endpoints

Method Endpoint Description
POST /v1/pastes Create new paste
GET /v1/pastes List recent pastes
GET /v1/pastes/{id} Get paste by ID
DELETE /v1/pastes/{id} Delete paste
